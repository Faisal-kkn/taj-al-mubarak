# Bustion - Changelog

All notable changes to the Bustion template will be documented in this file.

## [1.0.0] - 2025-04-21

### Added

- Initial release of Bustion template
- Complete set of HTML pages:
  - Home page
  - About page
  - Services pages
  - Blog pages
  - Contact page
  - FAQ page
  - Team page
  - Career page
  - Bus detail pages
  - 404 page
- Responsive design implementation
- Documentation
- Cross-browser compatibility
- Modern UI components and features

### Features

- Built with Bootstrap 5.3
- Fully responsive layout
- Modern and clean UI design
- Multiple page layouts
- Font Awesome icons integration
- Google Fonts integration
- SEO optimization
- Fast loading speed
- Easy customization options
