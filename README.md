# Bustion - Bus Transportation HTML Template

Bustion is a modern, responsive HTML template designed specifically for bus transportation and travel services. Built with Bootstrap 5.3, it features a clean and professional design with multiple page layouts and components.

## Features

- 🚌 **Modern Design**: Clean and professional layout optimized for bus transportation services
- 📱 **Fully Responsive**: Works perfectly on all devices and screen sizes
- 🎨 **Multiple Pages**: Comprehensive set of pages for all your needs
- ⚡ **Fast Loading**: Optimized for performance and speed
- 🎯 **Cross-browser Compatible**: Works flawlessly on all modern browsers
- 🛠️ **Easy to Customize**: Well-documented and easy to modify
- 📦 **Bootstrap 5.3**: Built with the latest Bootstrap framework
- ✨ **Modern UI Components**: Includes various modern UI elements
- 🎨 **Customizable Colors**: Easy color scheme customization

## Installation

1. Download the template package from ThemeForest
2. Extract the downloaded zip file
3. Open the extracted folder
4. Navigate to the `HTML_Template` folder
5. Start editing the files using your preferred code editor

## Customization

### CSS Customization

The template uses CSS variables for easy customization. You can find these in the `style.css` file:

```css
:root {
  --text-color-1: #47464d;
  --text-color-2: #a6a5aa;
  --heading-color: #201c30;
  --color-main: #3f1ec6;
  --color-darker: #22156b;
  --color-lighter: #7351ff;
  --color-subtle: #ff9203;
  --color-border: #bebcc7;
  --color-accent: #f4f4f4;
  --color-error: #e22d2d;
  --color-warning: #e2d52d;
  --color-info: #2663cf;
  --color-success: #20bd3c;
  --font-1: "exo-2";
}
```
